# Library Management System - Complete (No Login)

This project includes Home, Books, Members, Transactions (manual entry), and Reports.

## Run
- Open in IntelliJ (File -> Open -> select `pom.xml`)
- Set Project SDK to Java 17
- Run `LibraryManagementCompleteApplication`
- Visit http://localhost:8080

## Notes
- Transactions use manual entry: enter member name and book title in the borrow form.
- If the book title exists in the database, the app will mark its availability accordingly.
- Reports page shows basic counts.
